# DoorDasH: Scare vs Laugh Touchdown

A JavaFX board game inspired by Monsters Inc.

## Features
- Choose SCARER or LAUGHER
- 100-cell zigzag board
- Door, Card, Monster, Conveyor Belt, and Contamination Sock cells
- Dice-based movement
- Monster-specific powerups
- Dynamic energy, status effects, shields, freeze, confusion, and role swapping
- Card effects including energy steal, start over, shield, swap, and confusion
- Game-over screen and evaluation cheat buttons

## Technologies
- Java
- JavaFX
- Object-Oriented Programming
- MVC-style GUI integration
